# VS Code Productivity and Keyboard Shortcuts

## Useful VS Code Shortcuts

### Navigation
1. **Command Palette**  
   Cmd + Shift + P / Ctrl + Shift + P  
   Allows quick access to all VS Code commands.

2. **Quick File Open**  
   Cmd + P / Ctrl + P  
   Opens files quickly by typing the file name.

3. **Go to Line**  
   Ctrl + G  
   Jumps directly to a specific line number.

---

### Editing
4. **Multiple Cursors**  
   Option + Click / Alt + Click  
   Edit multiple places in the file at the same time.

5. **Select All Occurrences**  
   Cmd + Shift + L / Ctrl + Shift + L  
   Selects all occurrences of a word for quick editing.

6. **Duplicate Line**  
   Shift + Option + Down / Shift + Alt + Down  
   Duplicates the current line.

7. **Move Line Up or Down**  
   Option + Up/Down / Alt + Up/Down  
   Reorders lines without cutting and pasting.

---

### Debugging
8. **Toggle Breakpoint**  
   F9  
   Adds or removes a breakpoint.

9. **Start or Continue Debugging**  
   F5  
   Starts the debugger or continues execution.

---

### Terminal
10. **Toggle Integrated Terminal**  
    Ctrl + `  
    Opens or closes the built-in terminal.

---

## Reflection

### Which shortcuts will be most useful in daily work?
Multiple cursors, quick file open, and the command palette will be the most useful because they save time and reduce repetitive actions.

### How can keyboard shortcuts improve productivity?
Keyboard shortcuts reduce reliance on the mouse and speed up common tasks. This helps maintain focus and allows developers to work more efficiently, especially in larger projects.

---

## Conclusion
Learning and using VS Code keyboard shortcuts improves workflow efficiency and helps developers work faster and more consistently.
